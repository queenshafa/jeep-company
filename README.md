# Jeep Website Remake

A front-end development project focused on UI slicing using Bootstrap and SCSS. The primary goal of this project is to practice responsive layout construction, modular 
styling with SCSS variables, and implementing modern automotive web design patterns.

## 🚀 Sections
* **Hero Section:** Featuring Jeep's tagline, Jeep Indonesia's short description, navigation bar, and a really cool advertisement video.
* **About Section:** Big texts about what Jeep is.
* **Models Section:** Showing 4 Jeep's big models so the user can view what Jeep sells.
* **Dealership Section:** Find Jeep Dealerships nearby easily.
* **Jeep Life Section:** Find additional features/programs provided by Jeep.
* **Jeep Ownership Section:** A form for users to fill out if they want to get more news from Jeep.
* **Footer Section:** A footer consists of Jeep's tagline, Jeep's website pages list, Jeep's social media, and Customer Service.

## 🛠️ Tech Stack
* **Structure:** HTML5
* **Styling:** [Bootstrap 5](https://getbootstrap.com/) & [SCSS](https://sass-lang.com/)
* **Design:** Figma
